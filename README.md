# PSU_2019_malloc
Recreate Malloc-Like Functions.

Allocation Strategy: Malloc Buddy.

# Subject:
Using only brk/ sbrk, rewriting the 5 following C library functions:
 - malloc
 - calloc
 - realloc
 - reallocarray
 - free

#Build
(via Makefile)
 - make / make re:\
    Create Shared Library containing functions.

 - make fclean:\
    Clean Objects Files
